API : Header Parse Microservice

This is a basic API that tells you Three details about user.

  • IP Address
  • Language
  • Software

Example :

Request to : 127.0.0.1:3000/api

Returns a JSON object as such :

{
'ipaddress' : '117.21.55.43' ,
'lang' : 'en-US',
'software' : 'x11; Linux X84_64'
}
NOTE : If failed to determine IP address then IP of local host will be returned i.e 127.0.0.1
